A compact label for status, metadata, refs, and counts.

```jsx
<Badge tone="success" dot>Ready</Badge>
<Badge tone="warning" dot>Building</Badge>
<Badge tone="accent" icon="fa-solid fa-code-branch">main</Badge>
<Badge tone="cyan" shape="square">MD</Badge>
```

Tones: `neutral`, `accent`, `cyan`, `success`, `warning`, `error`. Use `dot` for live status, `icon` for a Font Awesome glyph, `shape="square"` for file-kind tags.
