A text or search field. Boxed by default; use `flush` for the borderless sidebar search that reads as part of the chrome.

```jsx
<Input icon="fa-solid fa-magnifying-glass" type="search" placeholder="Search docs…" flush />
<Input placeholder="Project name" />
```

Pass any native input props (`value`, `onChange`, `disabled`, `type`). `icon` takes a Font Awesome class string.
